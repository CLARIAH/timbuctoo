Solr can be configured as follows:
- create a directory 'solr' in ${user.home}/repository on the server;
- create a directory 'conf' in the 'solr' directory.;
- copy the contents of this directory to the 'solr' directory;
- refer to the 'solr' directory in the configuration file;
- specify the cores in the 'indexeddoctypes' entry of the configuration file.
